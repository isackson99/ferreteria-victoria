Configuración de Impresión

info Cómo configurar tu impresora térmica

El sistema usa la impresora predeterminada de Chrome.

  1. Pulsa Imprimir ticket de prueba abajo
  2. En el diálogo de Chrome, selecciona tu impresora térmica
  3. Ajusta el tamaño de papel a 80mm × Auto (o "Sin márgenes")
  4. Desde ese momento Chrome recordará la impresora seleccionada
warning Impresión sin diálogos — Chrome Kiosk

Para que Chrome imprima directamente a la impresora por defecto sin mostrar ningún diálogo, debe abrirse con el flag --kiosk-printing.

  1. Crea un acceso directo a Chrome en el escritorio
  2. Haz clic derecho → Propiedades
  3. En el campo Destino, reemplaza el contenido por:
{{ comandoKiosk }}

Usa ese acceso directo para abrir Chrome en esta PC. Puedes configurar la impresora por defecto del sistema en Configuración de Windows → Bluetooth y dispositivos → Impresoras.
